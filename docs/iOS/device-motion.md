# Device motion

This section groups the Core Motion APIs that deliver sensor-fused device motion data.

## Included items
- Getting processed device-motion data: Access motion values that have been corrected for environmental bias (for example, gravity).
- CMDeviceMotion: A container for attitude, rotation rate, and acceleration measurements.
- CMAttitude: The device orientation relative to a known reference frame at a specific time.
- CMAttitudeReferenceFrame: The available reference-frame options used for attitude reporting.
- CMHeadphoneMotionManager: The manager for motion updates from supported headphones.
